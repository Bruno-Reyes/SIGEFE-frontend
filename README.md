# Instrucciones de instalacion FRONTEND

## Instalacion

* Instalar NodeJS o BunJS

NodeJS -> https://nodejs.org/en/

BunJS -> https://bun.sh/

De preferencia BunJS

* Una vez lo tengas instalado puedes confirmar tu instalaci√≥n con: 
```bash
# Para NodeJS
node --version
# Para BunJS
bun --version
```

* Instala todas las librerias con:

```bash
# Para NodeJS
npm install
# Para BunJS
bun install 
```

* Si ya lo tienes instalado y te pide una libreria nueva, solamente vuelve a ejecutar el comando anterior

## Ejecucion

* Usa el comando:
```bash
# Para NodeJS
npm run dev
# Para BunJS
bun run dev 
```
